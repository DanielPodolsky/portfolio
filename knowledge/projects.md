## Tell me about MentorHIT

### Also: MentorHIT, mentor hit, mentoring project, AWS hackathon, hackathon project, first place

AI mentoring platform I built with a team of four during a hackathon. I handled full-stack - React frontend, backend API, and AWS integration. Uses Bedrock LLMs with RAG via OpenSearch so students get personalized career guidance based on their profile and job market data. Won first place at HIT, second place nationally at AWS Public Sector Hackathon. College is reviewing it for official integration.

## Tell me about the Log Analyzer

### Also: LLM Log Analyzer, log analyzer, logging project, anomaly detection, DevOps project

Intelligent system that automates log anomaly detection and root-cause classification. Containerized microservices: React frontend, Node.js analyzer, Python LLM service. Built a factory pattern for LLM provider abstraction - swap between Hugging Face and OpenAI without code changes. Auto-detects critical issues and sends actionable recommendations via email alerts.
